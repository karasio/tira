Tehtäväkuvaus: Toteuta binäärihakuun (=puolitushaku) perustuvat avaimella haku- ja lisäysoperaatiot. Määrittele luokka, joka kapseloi avaimet tallettavan taulukon ja tarjoaa em. julkiset operaatiot.
